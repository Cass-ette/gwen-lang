# Gwen 语言自检测试套件

> 业界标准算法测试，验证 Gwen 实现正确性

## 测试层级

| 层级 | 目的 | 测试数量 |
|------|------|----------|
| L1 基础 | 语法、运算、控制流 | 10 |
| L2 数据结构 | list、索引、迭代 | 8 |
| L3 算法 | 标准算法实现 | 12 |
| L4 类型系统 | 精度、溢出、转换 | 12 |
| L5 边界 | 极端输入、边界值 | 12 |

## 通过标准

- **全部通过**：语言实现正确
- **L1/L2 失败**：核心语法有 bug，必须修复
- **L3 失败**：某些特性不支持，可降级为示例
- **L4 失败**：类型系统待完善
- **L5 失败**：优化空间，不影响正确性

## 运行方法

```bash
# 全部自检
python -m gwen tests/self_check/run_all.gw

# 单层
python -m gwen tests/self_check/l1_basic.gw
python -m gwen tests/self_check/l3_algorithms.gw
```

## 测试编写规范

1. **独立性**：每个测试不依赖其他测试状态
2. **确定性**：相同输入必得相同输出
3. **可验证**：输出包含 `PASSED` 或 `FAILED` 标记
4. **覆盖性**：尽量覆盖语言 spec 的所有特性
5. **边界值**：包含空输入、极值、异常路径
